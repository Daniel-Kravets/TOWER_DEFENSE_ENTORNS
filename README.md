# Forest Guardians

Microvideojoc tower defense de fantasia en 2D i vista superior, desenvolupat amb LÖVE2D i Lua.

## Objectiu
Defensar el camí impedint que els enemics arribin al final del recorregut.

## Tecnologies
- LÖVE2D
- Lua
- Visual Studio Code

## Estructura inicial
- `main.lua`: punt d’entrada del joc
- `conf.lua`: configuració bàsica
- `src/entities/`: entitats del joc
- `src/managers/`: gestors del joc
- `diagrames/`: diagrames UML de la fase 2
